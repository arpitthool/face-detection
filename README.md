# Face Detection C++ App

A C++ application for detecting single or multiple faces (via webcam) using OpenCV.

## Requirements

- OpenCV 4.x
- CMake 3.10+
- C++11 compatible compiler (g++, clang++, etc.)

## Building

```bash
mkdir build
cd build
cmake ..
make
```

## Running

```bash
cd build
./face_detection
```
